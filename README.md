# Recipes App

## 📖 About the project

Recipes App is an Android application developed in Kotlin that allows users to explore and save recipes.

## 🛠 Technologies

- Kotlin
- Android Studio
- Gradle
- Material Design

## 🚀 How to run the project

1. Clone the repository

git clone https://github.com/GustavoStorion/Recipes-App.git

2. Open in Android Studio
3. Sync Gradle
4. Run the project
